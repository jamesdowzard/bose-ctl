The Problem

Bose pushed a firmware update that removed physical button source switching on the QC Ultra headphones.

Before: hold button → cycle sources
After: open phone app → navigate menus → tap source

With 3 devices (MacBook, iPhone, S21) but only 2 multipoint slots, switching sources meant pulling out the phone every time.

Goal: control headphone sources from the Mac, no app needed.

The Approach

The Bose Music app communicates with headphones over Bluetooth. If we can understand the protocol, we can replicate it.

The Protocol

Every message follows a 4-byte header + payload structure:

Block0x00-0x06
Function0x00-0x0F
OperatorGET/SET/...
LengthN bytes
Payloadvariable
Operators
0x01 GET    0x03 RESPONSE    0x04 ERROR    0x05 START    0x06 SET    0x07 ACK

Same format as QC35 (2017), but CONNECT uses operator 0x05 (START) instead of 0x06 (SET).
